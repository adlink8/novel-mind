# NovelMind GSD Project

## Core Value

NovelMind 是 AI 辅助小说理解与同人创作平台。当前 GSD 接管的核心价值不是新增 RAG，而是先建立可信实现基线。

## Onboarding Source

- README.md
- docs/需求文档.md
- docs/技术架构.md
- docs/路线图.md
- docs/项目状态.md
- docs/待办清单.md
- docs/竞品调研报告.md
- 实际代码：backend、frontend、docker-compose.yml

## Current Reality

代码实际状态是 PARTIAL。小说上传、分章、基础 CRUD、AI 模型后端 CRUD 和前端页面骨架存在；大量 AI 分析、RAG、图谱、时间线、同人文能力仍是占位或缺失。

## Key Decision

第一个 active milestone 固定为“审计与启动修复”。后续功能开发必须在该 milestone 验证通过之后再启动。
