# NovelMind - AI 辅助小说创作与理解平台

## 项目愿景

NovelMind 是一款面向小说爱好者和同人创作者的 Web 应用，通过 AI 深度理解小说内容，帮助用户梳理剧情、构建人物关系图谱、可视化时间线，并基于原作风格续写同人小说。

**一句话描述**：让 AI 成为你的小说伙伴 —— 读懂故事、理清脉络、续写篇章。

## 目标用户

- 长篇小说读者（需要理清复杂剧情和时间线）
- 同人小说/同人文创作者（需要基于原作风格续写）
- 书评人和内容创作者（需要深度分析小说结构）
- 写作爱好者（希望从优秀作品中学习写作技巧）

## 核心价值

1. **深度理解**：AI 不只是摘要，而是真正理解人物动机、伏笔、叙事结构
2. **可视化呈现**：将复杂的时间线和人物关系变成直观的图表
3. **风格续写**：学习原作风格、用词习惯、叙事节奏来续写同人文
4. **开放生态**：支持多种 AI 模型，用户自由选择

## 技术栈

详见 [技术架构.md](./技术架构.md)。当前实现基线：
- 前端：Next.js 16 (App Router) + TypeScript + Tailwind CSS + shadcn/ui
- 后端：FastAPI + SQLAlchemy async + Pydantic v2
- 数据：PostgreSQL 16 + pgvector + ChromaDB
- AI：LiteLLM 1.83.10+ 统一封装

## 核心功能模块

### 1. AI 剧情理解与分析
- 导入小说文本（支持 TXT、EPUB、PDF）
- AI 自动生成剧情摘要（章节级、卷级、全书级）
- 人物识别与性格分析
- 伏笔/悬念/转折点的智能标注
- 叙事手法分析（视角、时间线结构、叙事节奏）
- 基于语义的剧情搜索（"哪一章主角第一次遇到XX"）

### 2. AI 续写/同人文创作
- 选择原作风格参数（文风、用词偏好、叙事视角）
- 设定续写方向（时间线分支、新角色、新场景）
- 章节级续写，支持迭代修改
- 风格一致性评分
- 多人协作编辑
- 导出为 TXT/Markdown/EPUB

### 3. 时间线梳理与可视化
- AI 自动提取小说中的时间标记和事件
- 生成可交互的时间线视图
- 支持多视角时间线（不同角色的经历线）
- 平行时间线对比
- 事件因果链可视化
- 手动编辑和调整时间线

### 4. 人物关系图谱
- AI 自动识别人物并构建关系网络
- 关系类型标注（亲属、敌友、师徒、恋人等）
- 关系演变追踪（按章节/卷展示关系变化）
- 角色成长轨迹
- 人物卡片（性格、能力、经典台词、出场统计）

## 非功能性需求

- **性能**：首页加载 < 2s，API 响应 < 500ms（非 AI 请求）
- **可用性**：支持移动端响应式布局
- **数据安全**：小说内容加密存储，用户数据不共享
- **国际化**：初版支持中文，后续扩展英文/日文

## v1 与 v2 范围

### v1（MVP）
- 小说导入（TXT 格式）
- 单本小说的剧情分析与摘要
- 基础时间线生成与展示
- 基础人物关系图谱
- AI 续写（单章节）
- 支持 OpenAI + 1个备选模型

### v2
- EPUB/PDF 导入
- 多本小说关联分析
- 高级可视化（交互式、动画）
- 多人协作编辑
- 多模型切换（3+ 模型）
- 导出功能
- 用户社区与分享

## 项目约束

- 单人/小团队开发
- 优先保证核心功能可用性
- 前端优先使用成熟组件库，减少自定义开发
- AI 调用成本需要可控（支持本地模型作为备选）
